MicroFocus Zones

Create strict focus capsules, gate unrelated tabs, and restore your task context in one click.

What this is

MicroFocus Zones builds task-scoped "mini desktops" in your browser. Each Zone captures the exact tabs you need for one task, hides or dims everything else, and adds a gentle gate when you drift.

Why? Traditional blockers punish domains; this leans on task relevance. A YouTube debugger video is fine in an ESP32 task, but random music videos aren't.

Use it for: deep work sessions, research with many tabs, writing, coding spikes, or any task where context switching kills flow.

Core ideas: task capsules (saved tabs + notes), semantic-ish gates (ask "is this related?"), and micro-focus timers (25/50) to keep you honest without hard bans.

Quick start
1) Pin the icon
Open chrome://extensions -> enable Developer Mode -> pin "MicroFocus Zones" in your toolbar.
2) Open the side panel
Click the toolbar icon; the side panel opens automatically.
3) Create a Zone
Add a task title + notes, then hit "Create Zone".
4) Capture your current tabs
With relevant tabs open, click "Capture tabs" on that Zone to save the capsule.
5) Enter the Zone
Click "Enter Zone". Unrelated pages will get a gate prompt.
6) Run a focus timer (optional)
Start a 25m or 50m timer; the status shows on the live pill.
7) Use the gate
When an unrelated page appears, choose: let through (research), 5-min break, mark off-task, or keep blocked.
8) Exit
Use "Exit Zone" to end; history shows session stats.
Tips
  • Saved capsules remember URLs/titles (and attempt scroll positions); capture again anytime to refresh.
  • Chrome's special pages (chrome://) cannot be gated (content scripts aren't allowed there).
  • If the icon doesn't open the panel, update Chrome to the latest version.
